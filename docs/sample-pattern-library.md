# 样本模式库

这份文档不是文章合集，而是从历史样本里提炼出来的“可复用模式”。

目标是教模型什么叫“像作者写”，而不是给模型一个整篇照抄的素材库。

---

## 1. 开头模式

### 模式 A，先抛反常识结论

适用：

- 价值重估型
- 误解纠偏型

写法：

- 上来先否定市场标签
- 再给出新的定义

示意：

- 真正的前滩南，不是谁离前滩最近，而是谁能承接前滩，却又不与它同价
- 这个板块最容易被误解的地方，不是它看起来破，而是大家根本没看懂它的空间位置

模型要学的不是句子本身，而是：

- 开头先给判断
- 一上来就把问题讲尖锐

### 模式 B，先抛供应/断供焦虑

适用：

- 断供型
- 规划拆解型

写法：

- 用一个“看起来很大”的结论把读者拉住
- 再回到控规和储备用地解释

示意：

- 看完规划我发现康桥也要断供了
- 上海滨江住宅要断供了？

模型要学的是：

- 先给一个供给结论
- 后面再用地块、控规和供地解释，不要倒过来

### 模式 C，先用情绪切入，再迅速落回结构

适用：

- 更新拆迁型

写法：

- 允许先用一句感受起手
- 但必须很快回到板块结构和更新逻辑

示意：

- 说实话，这是一个让人心情很复杂的板块

模型要学的是：

- 情绪只是入口，不是全文基调
- 不能一路感叹到底

## 2. 结构模式

### 模式 A，误解 -> 骨架 -> 片区 -> 结论

适合：

- 北蔡
- 康桥
- 三林

标准节奏：

1. 大家为什么会看错
2. 这个板块真实骨架是什么
3. 片区拆解
4. 最后回到价值判断

模型要学的是：

- 不要急着讲配套
- 先讲为什么这个板块会被误读

### 模式 B，标签重写 -> 标准重建 -> 验证 -> 收束

适合：

- 浦锦才是前滩南

标准节奏：

1. 市场旧标签是什么
2. 为什么旧标签不成立
3. 新标签的标准是什么
4. 用交通、资源、价格去验证
5. 最后给出新定位

模型要学的是：

- 重估型文章不是“夸”，而是“重建定义标准”

### 模式 C，规划拆解 -> 片区供地 -> 未来走势

适合：

- 周浦还有八千套新房
- 康桥也要断供了

标准节奏：

1. 先说总量
2. 再说总量里哪些有效、哪些无效
3. 分区讲供应结构
4. 最后讲未来会不会断档

模型要学的是：

- 不要只会报数字
- 供地文章最重要的是解释“这些地分别意味着什么”

## 3. 抓手式命名模式

抓手式命名是你文章最强的风格资产之一。

### 常见模式

- 灯下黑
- 散装
- 诸侯国
- 九宫格
- 平行世界
- 前滩南
- 后花园
- 造城记

### 模型该怎么学

模型不要复制这些词，而要学习：

- 一个板块最好有一个“记忆钩子”
- 这个钩子应该概括板块真实结构
- 钩子最好可以服务后文分区拆解

### 好的抓手满足什么

- 一听就懂
- 有画面
- 能概括结构
- 能服务论证

### 差的抓手是什么

- 只有噱头，没有结构意义
- 太长
- 太抽象
- 只是情绪，不是认知工具

## 4. 高光表达模式

### 模式 A，结构错配句

典型作用：

- 用一句话把表象和底层区分开

示意方向：

- 行政边界会骗人，但真实生活边界不会
- 板块标签会骗人，但房价地图不会

模型要学的是：

- 把“表面”和“底层”分开写
- 用一句话形成认知落差

### 模式 B，对比句

典型作用：

- 快速让读者感受到板块内部的割裂

示意方向：

- 左眼是 20 万+，右眼是 3 万+
- 一路之隔，咫尺天涯

模型要学的是：

- 用极短的对比，让板块内部差异立起来

### 模式 C，定义重写句

典型作用：

- 把一个老标签重新定义

示意方向：

- 真正的前滩南，不是……

模型要学的是：

- 重估型文章一定要有“重写定义”的一句

## 5. 分区拆解模式

作者不是按行政区写板块，而是按“真实生活单元”拆。

### 片区拆解时必须讲的 5 件事

1. 这个片区是什么
2. 它和其他片区最本质的区别是什么
3. 它的核心优势是什么
4. 它最大的短板是什么
5. 它适合哪类人

### 模型要避免的错误

- 只按地图平铺
- 每个片区都写成一样的模板化段落
- 每个片区都只有优点没有代价
- 不讲片区之间的关系

### 好的片区拆解应该让读者感受到

- 这个板块不是一个整体
- 每个片区的逻辑根本不同
- 买错片区，比买错板块更可怕

---

## 6. 结尾模式

### 模式 A，回到购房者判断

最常用，也最稳。

落点：

- 这个地方对谁成立
- 这个地方对谁不成立
- 它真正的门槛是什么

### 模式 B，回到变量

适合：

- 断供型
- 规划拆解型

落点：

- 未来真正要看的，不是口号，而是哪块地什么时候出来
- 不是今天热不热，而是供应会不会断档

### 模式 C，回到重估结论

适合：

- 价值重估型

落点：

- 这个板块真正该被放在什么位置理解

模型要学的是：

- 结尾不是抒情
- 结尾是重新落回现实判断

---

## 7. 给模型的最终提醒

你学习样本，不是为了复述样本，而是为了学会下面这些东西：

- 作者如何开头
- 作者如何纠偏
- 作者如何搭空间骨架
- 作者如何给片区起名字
- 作者如何在结尾把文章落回判断

如果只是把历史文章改写一遍，那就不是“像作者写”，而只是“像个检索器在拼稿”。
