# 样本标注目录

这份目录不是文章全文，而是把 `wz` 历史文章拆成可供数字人学习的结构化资产。

---

## 1. 北蔡滨江断供

- 标题：`上海滨江住宅要“断供”了？`
- 题型：断供型
- 题眼：滨江核心区住宅储备用地看起来还热闹，但真实供给已经接近结构性见底。
- 开头动作：先抛供给焦虑，再说明这不是日常供应波动，而是控规意义上的断供。
- 主判断：断供不是一句情绪话，而是不同滨江区段的储备用地、旧改顺序和开发节奏共同造成的结构结果。
- 结构骨架：按江水流向顺序拆区段。
- 现实落点：不同区段的供给先后顺序会决定未来价格预期和关注重点。
- 不可复用词壳：`顶流对决`、`全乐章最激昂的高潮`

## 2. 川沙拆迁

- 标题：`看完规划我发现川沙要拆这么多`
- 题型：价值重估型
- 题眼：川沙行政范围很大，但真正值得讨论的是核心城区与大面积留白之间的结构失衡。
- 开头动作：从历史身份和今天的空间形态反差切入。
- 主判断：川沙不是简单的“远”，而是一个被行政范围和土地约束共同塑造的瘦高型板块。
- 结构骨架：老城厢、华夏社区、城南社区等核心片区拆解。
- 现实落点：远期兑现潜力很大，但兑现需要发展决心和时间成本。
- 不可复用词壳：`远期支票`

## 3. 康桥断供

- 标题：`看完规划我发现康桥也要断供了`
- 题型：断供型
- 题眼：康桥的关键不是热度低，而是散装结构导致很多区域长期各自为政，供给也在不同子区域同时见底。
- 开头动作：先写“位置不差却没有存在感”的反差，再给“散装”判断。
- 主判断：康桥的断供不是一个整体板块的同频断供，而是散装结构下多个片区逐步见底。
- 结构骨架：别墅区、周浦西社区等子区域逐个拆。
- 现实落点：不同区域的价值完全不能一概而论，归属感和兑现节奏都不同。
- 不可复用词壳：`四川和重庆`、`十三太保`

## 4. 浦锦前滩南

- 标题：`看完规划我发现浦锦才是前滩南`
- 题型：价值重估型
- 题眼：市场把“前滩南”说成地理邻近词，但作者把它重写成“承接前滩但不与前滩同价”的结构定义。
- 开头动作：先否定被用烂的旧标签，再重建标准。
- 主判断：真正的前滩南不是最近，而是同时满足通勤、资源承接和价格断层。
- 结构骨架：标准建立 -> 三林滨江排除 -> 规划溯源 -> 距离与成本 -> 内部南北切分。
- 现实落点：浦锦成立，但它的成立伴随着拥挤、路网和界面上的真实代价。
- 不可复用词壳：`前滩南` 本身不能被默认套用到其他题。

## 5. 周浦新房供应

- 标题：`看完规划我发现 周浦还有八千套新房`
- 题型：更新拆迁型
- 题眼：周浦不是没有资源，而是历史中心身份、当前界面拧巴和未来供应之间存在张力。
- 开头动作：从“老大哥”身份和今天的拧巴感切入。
- 主判断：周浦的问题不是有没有配套，而是供给结构和城市界面长期没有顺起来。
- 结构骨架：西中东片区与万达周边供地拆解。
- 现实落点：刚需资源看起来齐，但居住界面和供应兑现并不轻松。
- 不可复用词壳：`小上海`、`拧巴`

## 6. 三林价差

- 标题：`如果问上海哪个板块的房价“贫富差距”最大？答案一定是三林。`
- 题型：价值重估型
- 题眼：三林最值得写的不是贵或便宜，而是夸张价差背后的边界和切割。
- 开头动作：用极端价差直接打读者。
- 主判断：三林不是一个统一板块，而是被高架、主干道和价格体系切成几个平行世界。
- 结构骨架：先讲认知错位，再搭九宫格地图，再分世界讲。
- 现实落点：不同世界对应完全不同的人群、价格和生活方式。
- 不可复用词壳：`宇宙中心`、`平行世界`、`九宫格`

## 7. 北蔡价值重估

- 标题：`说实话，这是一个让人心情很复杂的板块`
- 题型：价值重估型
- 题眼：北蔡的核心不是“乱”，而是顶级位置与低感知度之间的巨大错配。
- 开头动作：先用情绪起手，再迅速落回位置和结构。
- 主判断：北蔡的价值长期被低估，不是因为没资源，而是因为不同拼图之间的界面差异和认知错配。
- 结构骨架：大华、动迁天团、御桥、楔形绿地等几块拼图递进展开。
- 现实落点：不同拼图对应完全不同的人群、预算和持有周期。
- 不可复用词壳：`灯下黑`、`浦东阵眼`、`大华造城记`
