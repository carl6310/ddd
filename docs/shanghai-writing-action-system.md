# 上海板块写作动作系统

这份文档不是结构模板补丁，而是给生成链路加一层真正的“写作动作约束”。

它解决的问题只有一个：

同样是上海板块稿，为什么有些稿子只是旧结构重写一遍，有些稿子却像这次真的看见了一个新的地方。

答案不在提纲，而在动作。

---

## 1. 先分清两层东西

结构层负责回答：

- 先讲什么
- 后讲什么
- 哪一段讲空间骨架
- 哪一段讲片区拆解

动作层负责回答：

- 这篇最具体的新观察是什么
- 读者会在哪一下被拉住
- 中途哪里该故意打断
- 哪里要把地图翻成生活
- 哪里要升维
- 结尾回扣什么

没有动作层，文章再工整，也容易像旧稿换板块。

---

## 2. 这套系统的 10 个核心动作

### 2.1 新观察

每篇都必须先回答：

- 这篇最具体的新观察是什么
- 不是“这个板块被低估了”这种旧结论
- 而是“它内部哪一种错位、断裂、代价、切割，值得今天重新写”

新观察必须具体到可以落进一句话里。

### 2.2 推进节奏

一篇板块稿不能只是按模块摆信息。

默认推进方式应该是：

1. 先挑破误解或旧标签
2. 再搭地图，给出真实骨架
3. 再拆成几个不同的世界
4. 最后回到具体人群和现实代价

但“推进”不是“列目录”，而是每一段都要把读者往下一段推。

### 2.3 故意打破

当文章快滑进平推资料时，必须主动打断。

常见打破方式：

- 讲到一半，突然切进一个具体场景
- 刚讲完优势，马上补一个拧巴处
- 刚要升维，先刹车讲代价
- 用一句短句单独成段，强行收一下注意力

### 2.4 断裂句

每篇都最好有一句能单独站住的短句。

它不是标题党，而是正文里最该被记住的那一下。

例如：

- 这不是一个板块，这是几种人生路径挤在一起。
- 真正难的不是买不买，而是你买进的是哪一种等待。

### 2.5 私人姿态

作者不是全知上帝。

要允许文章里出现：

- 我更在意什么
- 我为什么会这样判断
- 我承认这是一种带体感的判断

这不是削弱判断，反而会让判断更像活人说出来的。

### 2.6 人物 / 生活场景

地图必须翻译成生活。

至少落一次：

- 早高峰怎么走
- 接娃会不会别扭
- 从地铁站出来第一眼看到什么
- 首改买家会在哪一步犹豫

没有这一步，结构再完整，也只是板块百科。

### 2.7 文化升维

不要只在板块内部打转。

合格的升维是把这个板块放进更大的参照物里：

- 上海城市梯度
- 核心区外溢承接
- 存量更新逻辑
- 居住代际切换
- 认知滞后与定价修复

升维不是摆知识，而是让读者知道“为什么这件事不只属于这个板块”。

### 2.8 回环呼应

结尾不能平着收。

结尾要做的不是重复总结，而是回扣前文埋下的东西：

- 回扣开头的误解
- 回扣中段那句断裂句
- 回扣那个具体人物
- 回扣那种代价感

最好的结尾会让读者意识到，原来前面那句判断在这里闭环了。

### 2.9 读者直呼

正文里要知道自己在跟谁说话。

不要写“购房者”这种空词，要更具体：

- 首改
- 总价卡死的改善
- 被前滩叙事吸走的人
- 想赌规划兑现但又住在当下的人

点名之后，判断才会真正落地。

### 2.10 现实代价感

任何“值不值得看”都必须配一层代价。

要写清：

- 等待周期
- 总价门槛
- 界面妥协
- 流动性风险
- 产品力代价

没有代价感，文章就会自动向软文滑。

---

## 3. 每篇开写前必须先答的 5 个问题

1. 这篇最具体的新观察是什么？
2. 这篇最值得记住的一句话是什么？
3. 这篇要落到哪个人物或生活场景？
4. 这篇更大的上海参照物是什么？
5. 这篇最后要让读者意识到什么代价？

这五个问题答不清，宁可不写，不要硬跑结构。

## 4. 接进当前工作流的方式

### 4.1 ThinkCard

不是先冲着标题写，而是先输出：

- 素材吃透摘要
- 选题值判断：`strong / rework / weak`
- HKR
- 改方向建议
- 替代角度
- AI 角色边界

### 4.2 StyleCore

风格核不再只是旧动作卡，而是固定拆成：

- 节奏推进
- 故意打破
- 知识顺手掏出来
- 私人视角
- 判断力
- 对立面理解
- 情绪递进
- 亲自下场
- 人物画像法
- 文化升维
- 句式断裂
- 回环呼应
- 谦逊铺垫法
- 现实代价

### 4.3 提纲生成

每一段都不只写“讲什么”，还要写：

- 这段执行什么动作
- 这段在哪里故意打破
- 它怎么把读者带到下一段
- 这段要兑现哪一项风格目标

### 4.4 正文生成

正文必须兑现动作卡，至少做到：

- 一次人物 / 场景落地
- 一次文化升维
- 一次现实代价提醒
- 一句短促断裂句
- 一个明确回环

### 4.5 VitalityCheck

除了查结构、证据、AI 味，还要查：

- 有没有新的具体观察
- 有没有私人姿态
- 有没有人物体温
- 有没有升维
- 有没有回环
- 有没有现实代价

这张卡不过线时：

- 正文可以继续保留和修改
- 但不能进入发布前整理

---

## 5. 最后的判断标准

一篇稿子是否过线，不只看它是不是“像你”。

还要看它是不是这一篇才成立。

也就是说，它必须同时满足：

- 有作者的判断方式
- 有这一次的新观察
- 有这一次的生活体感
- 有这一次的更大参照物
- 有这一次的结尾闭环

只有这样，文章才不会只是旧稿复写。
